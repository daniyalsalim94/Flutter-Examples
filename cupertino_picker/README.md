# cupertino_picker

A new Flutter application which implements CupertinoPicker

## How to implement a CupertinoPicker

- Use the following code to implement a CupertinoPicker

```

          CupertinoPicker(
            itemExtent: 50,
            onSelectedItemChanged: (int index) {

            },
            children: <Widget>[
              Text(
                "Text 1",
                style: TextStyle(
                  fontSize: 30.0,
                  color: Colors.red
                ),
              ),
              Text(
                  "Text 2",
                style: TextStyle(
                    fontSize: 30.0,
                    color: Colors.red
                ),
              ),
              Text(
                "Text 3",
                style: TextStyle(
                    fontSize: 30.0,
                    color: Colors.red
                ),
              ),
              Text(
                  "Text 4",
                style: TextStyle(
                    fontSize: 30.0,
                    color: Colors.red
                ),
              ),
            ],
          )

```
